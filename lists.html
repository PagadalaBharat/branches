FORMATTING TAGS
  • Marked Text: Highlight text using the mark tag. Example:Do not forget to buy milk today.
  • Italic Text: The i tag defines text in an alternate voice or mood, typically displayed in italic. Example: This text is italic
  • Emphasized Text: The em tag emphasizes text, often used for stress or importance. Example: This text is emphasized
  • Deleted Text: The del tag indicates deleted text (usually with a strikethrough). Example: My favorite color is blue red.
  • Inserted Text: The ins tag defines inserted text (usually underlined). Example: My favorite color is blue red.
  • Smaller Text: The small tag defines smaller text. Example: This is some smaller text.
  • Bold Text: Use the b tag or the strong tag to make text bold. Example: This text is bold
  • Subscript Text: Use the sub tag for subscript (half a character below the normal line). Example: This is subscripted text.
  • Superscript Text: The sup tag defines superscript (half a character above the normal line). Example: This is superscripted text.
  • Code Fragment: To display a fragment of computer code, use the code tag. Example: console.log('Hello, world!');
  • Deleted Text: The del tag indicates deleted text (usually with a strikethrough). Example: My favorite color is blue red.
BLOCK ELEMENTS
A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
INLINE ELEMENTS
An inline element does not start on a new line.
